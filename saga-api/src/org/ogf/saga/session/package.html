This package contains the {@link org.ogf.saga.session.Session Session} interface, which provides a common interface for managing security information (by storing {@link org.ogf.saga.context.Context Context} objects). Sessions get created by the factory methods {@link org.ogf.saga.session.SessionFactory#createSession() SessionFactory.createSession()} and {@link org.ogf.saga.session.SessionFactory#createSession(boolean) SessionFactory.createSession(boolean)}. There is a default session, which is used in all factory methods where a SAGA object is created without explicitly specifying a session.

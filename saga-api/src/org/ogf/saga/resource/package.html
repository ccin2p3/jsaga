This package provides classes to manage resources on remote grid/cloud systems.

This API (named Resource Management API or RM-API) is an extension to the SAGA API. It allows to acquire (create or get) and release (delete) resources. Resources are of three types:

  • {@link org.ogf.saga.resource.instance.Compute Compute} resources: to submit jobs
  • {@link org.ogf.saga.resource.instance.Storage Storage} resources: to store data
  • {@link org.ogf.saga.resource.instance.Network Network} resources: to link compute(s) and storage(s)

Acquiring, listing and releasing resources are operated by the {@link org.ogf.saga.resource.manager.ResourceManager ResourceManager}.
